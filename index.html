Energy Flexibility KPIs

Description

Python package used to calculate baseline-required and data-driven energy flexibility KPIs.

The package implements the KPIs introduced in [1] using the EFOnt semantic ontology for building energy flexibility applications [2].

Contributing

energy_flexibility_kpis is an open-source project that continues to benefit from community-driven updates and suggestion. Please, visit the energy_flexibility_kpis to join the community and start contributing!
