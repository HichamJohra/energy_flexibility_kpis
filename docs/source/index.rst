Description
***********

Python package used to calculate baseline-required and data-driven energy flexibility KPIs. 

The package implements the KPIs introduced in :cite:p:`LiArxiv221112252` using the EFOnt semantic ontology for building energy flexibility applications :cite:p:`LI2022100113`.

Contributing
